CSS3
=> Cascading Style Sheet : 폭포수
=> 모든 속성 => 상속 베이스

3가지 방법
1)인라인 스타일
2)내부 스타일
3)외부 스타일 (*실무)

CSS선택자
1)전체선택자
-wildcard 선택자
-reset css
- *  == all

2)태그선택자
-특정한 태그를 선택해서 스타일 적용

3)id선택자
-id속성값을 가지고 있는 태그에게 스타일 적용
-"#"
-id 속성값은 반드시 html에서 1번만 사용해야함(js에서 오류)

4)class선택자
-class속성값을 가지고 있는 태그에게 스타일 적용
-"."
-class 속성값은 html에서 횟수 제한없이 사용가능 
-반복적으로 공통된 스타일 정의하고 적용


5)후손선택자
-부모 아래에 들어오는 모든 자식요소들
-자손선택자도 모두 포함
-"띄어쓰기"

6)자손선택자
-부모의 직계 자식요소
-">"

7) 속성선택자
-동일한 html태그의 동일한 속성을 가지고 있는 경우, 안에 들어가있는
속성값에 따라서 스타일을 상이하게 적용
ex)
input[type="text"] {
        background-color: red;
      }

8)동위선택자
- 동: 같을 "동"
- 위: "위"치
>같은 위치에 속해있는 요소들 간 스타일을 적용하고 싶을 때
  => 반드시 형제 요소
>특정 요소의 바로 뒤에 존재하는 형제를 찾아올때: "+"
#same-position > h1 + h2 // h1뒤에 있는 h2 하나만 찾아옴

>특정 요소의 뒤에 있는 모든 형제를 찾아올때: "~"
#same-position > h1 ~ h2 // h1뒤에 있는 h2 전부를 찾아옴


9)구조선택자
1) 일반 구조 선택자
1개의 부모 요소안에 있는 모든 형제들을 형태와 관계없이 입력된 순서대로
번호를 카운팅한다.
nth-child(n)
first-child
last-child
홀수 혹은 짝수번째 요소들만 찾아올 수도 있다.
2n(짝수)
2n + 1(홀수)

2) 형태 구조 선택자
1개의 부모 요소안에 동일한 형태를 가지고 있는 형제들 가운데서 순서대로
번호를 카운팅한다.
nth-of-type(n)
first-of-type
last-of-type

10)반응선택자
-사용자가 어떤 액션(*이벤트)을 실행 => 반응을 일으킨다.(마우스 오버 / 클릭 등)
- :hover: 마우스 오버
- :active: 클릭

11)상태선택자
-사용자의 이벤트 액션을 통해서 특정 태그 및 요소들의 상태가 변화되었다는 것을 감지하고 이를 통해서 스타일을 적용
-form : 버튼 / Check / radio 


12)링크선택자
> 1) 링크를 가지고 있는 요소들을 선택하기 위한 목적
 :link
> 2) 특정 링크가 클릭된 경우를 선택하기 위한 목적
 :visited

13)문자선택자
- 시작문자선택자
>특정 문서안에 시작하는 첫번째 단어 혹은 문장을 선택해서 
스타일을 적용하고자 할 때
::first-letter :시작하는 첫번째 문자열
::first-line : 시작하는 첫번째 행

- 전후문자선택자(*중요)
>문서상에 특정 요소의 앞과 뒤에 어떤 스타일링을 적용하고자 할 때 사용할 수 있는 선택자
>2차원적인 영역만 이야기하는 것이 아님 
>3차원적인 측면(*이미지 혹은 아이템을 기준으로 z축으로 양수/음수)
>스타일을 적용할 때 반드시 content: ""를 써야함
>position 속성: 부모 relative / 자식 apsolute
 부모요소의 전체 너비와 높이를 기준으로 
>전후문자 선택자의 모체가 되어지는 요소는 부모요소
>전후문자 선택자를 활용해서 탄생되는 요소는 자식요소

*실무에서 전후문자선택자를 프로페셔널하게 사용하는 방법
 => html을 통해서 속성을 정의할때 
 => data- :접두어 + 문자열 => 고유한 속성

*data- 접두어를 활용해서 어디까지 프로그래밍 혹은 스타일 (jquery)

-반응문자선택자
>사용자의 이벤트에 반응하는 문자 선택자
>반응선택자: 문자를 대상x
>반응문자선택자: 문자를 대상o

14)부정선택자
- 선택한 해당 요소를 제외한 나머지 요소들에게 스타일을 적용하고 싶은 경우 사용
 > not() => 해당 함수안에는 부정하고자 하는 기준값이 매개변수 포함될 수 있음
