CSS 속성------------------------------------
1) 단위: px / rem / em / % / vw / vh
> em: element약자. 부모요소의 font size를 기준으로 몇 배수의 크기를 가져가게 할 것인가.(부모 자식관계가 복잡해질수록 쓰기 어려움)

> *rem: root element약자. 현재 작업중인 문서 내 최상위 요소의 태그 혹은 그에 준하는 선택자 

실무:
우리 작업 제품(*웹페이지)
- default: 18px 
- * || body => font-size: 18px
- 9px = 0.5rem
- 1.5rem = 27px
(디폴트 값을 정하고 프로젝트 시작)
